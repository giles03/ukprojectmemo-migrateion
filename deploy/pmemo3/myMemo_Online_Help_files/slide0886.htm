ONLINE HELP AND TRAINING
Searching for existing projects
TIP – Searching for a fragment of the Artist name is safer as you wont need to know whether the surname or forename comes first. Searching for “jackson” for example will find “Jackson, Michael” as well as “Michael Jackson”.
You can search by a variety of attributes to find a Project
You will be able to search all projects within your own label group (unless you are in a support team and have access to more than one label)
If you have Edit or Create rights you will be able to update and submit any projects within your Label Group.
TIP – Search for a fragment of the Artist, Title, Cat Number etc and the system will return all projects matching that pattern. For example searching for “di” under artist will return Dido, Celine Dion, Cyndi Lauper etc.
