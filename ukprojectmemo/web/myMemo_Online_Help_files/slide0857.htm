ONLINE HELP AND TRAINING
How can I view a project memo?
TIP – Opening a project in View mode ensures that you can review the details of a project without locking it for other users
Clicking the View button for any project (using Search, My Recent Projects or email Notifications) will open the project in a read only page
Scroll down the page to see the details for each product, and use the Dashboard link to view the dashboard or the Edit button to enter Edit mode for that project.
