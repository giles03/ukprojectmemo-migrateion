ONLINE HELP AND TRAINING
The system will lock any project that has been opened for Editing but not yet submitted
•
To unlock a locked project:
•If the project is one that you are editing it will be available in your My Locked Projects queue on the welcome page. You can choose to cancel any changes since the last time the project was submitted or alternatively continue editing the project and submit the new version.
•This will also unlock the project and make the changes visible to everyone else with access to that UK Label Group.
•If the project is being edited by someone else contact the user and ask them to unlock or submit the project.
How do I edit a locked project?
Project locked by Simon. Only Simon (or the IT helpdesk) can unlock the project
Projects locked by you. Only you can unlock the project by clicking the link to continue editing and submit the updated project.
 If you choose Cancel & Unlock changes made in this session will be lost. If you have not submitted the memo at least once the Memo reference will no longer exist.
