ONLINE HELP AND TRAINING
Using the dashboard (1/2)
Use Daily Dash Comments to comment on any updates not reflected in the dash e.g ‘Parts cleared today’ will let anyone else who views this project today know that the problem has been investigated and resolved.
You can access the dashboard by clicking any of the smileys on the welcome page and drilling into the summary dashboard for that project.
The Planning section is for products that have not yet been created in GRAS or locally scheduled in GLORES.
There will be 1 row on the dashboard for each product.
The summary report gives you an overall view of the 6 stages involved in production of the product
 
Clicking on any of the smileys in the summary dashboard will drilldown into the underlying detailed monitoring points
