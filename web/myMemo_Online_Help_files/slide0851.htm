How do I create physical formats? (1/2)
This is the physical formats screen.
Select your format from the dropdown list.
Enter your release date using the popup calendar.
Enter the catalogue number (if an international product).
Enter the local catalogue number (if required ie Westlife01).
Enter price Category/Line (FOC is free of charge).
Choose your packaging spec from the dropdown list; if non-standard please add  details into “comments”.
Select the number of components ie number of CDs/vinyl etc in pack.
